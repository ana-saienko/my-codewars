# My Codewars Solutions

##### Convert boolean values to strings 'Yes' or 'No'. #####
**8 kyu**

```javascript
function boolToWord( bool ){
  if(bool === true){
    return "Yes"
  }else{
    return "No"
  }
}
```

##### Reversed Strings #####
**8 kyu**

```javascript
function solution(str){
  return str.split('').reverse().join('');
}
```
